# BC4P Project Summary

## Vision
Building a 12,000+ acre sovereign agricultural corridor (**Posterity Cell One**) using high-end tech and Gīkūyū heritage.

## The Data Collection Platform
A MERN-based tool to collect structured qualitative and quantitative data under 5 pillars:
1. Cultural
2. Social
3. Economic
4. Environmental
5. Technical

## Key Features
*   No-password auth for contributors (Name+Phone).
*   Rich text and multi-media support (Cloudinary).
*   Admin dashboard for reviewing and filtering data.
